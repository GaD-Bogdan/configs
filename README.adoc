# My configs for git, vim and bash


Apply .bashrc:
```
mkdir apply_configs_tmp && \
git clone https://github.com/GaD-Bogdan/configs.git apply_configs_tmp && \
mv apply_configs_tmp/bash_configs/.bashrc ~/ && \
rm -rf apply_configs_tmp
```
